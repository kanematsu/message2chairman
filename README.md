# message2chairman

## メッセージ閲覧手順書

1. 準備するもの
	- メッセージカード（QRコード・会社ロゴ付き）
	- インターネット接続された iPhone（Safari 推奨）

---

2. QRコードを読み込む
    1. iPhone の カメラアプリを起動します。
    2. メッセージカードに印字された QRコードにかざします。
    3. 画面上部に「Safariで開く」という通知が表示されます。
    4. 通知をタップしてブラウザ（Safari）を開きます。

---

3. カメラアクセスの許可

    AR機能を利用するため、初回アクセス時に以下のポップアップが出ます。
	- 「カメラへのアクセスを許可しますか？」
    → “許可” をタップしてください。
    
    許可しない場合、AR表示が行えません。
    また、サイト個別の権限（アドレスバーの aA をタップ → Webサイトの設定 → カメラを許可）も確認してください。

---

4. 会社ロゴを読み込む（ARマーカー認識）
	1.	Safariで表示されたページが読み込まれたら、
        画面の指示に従い「NEOSロゴにカメラを向けてください」と表示されます。
	2.	メッセージカードに印刷された 会社ロゴをカメラに映してください。
	3.	ロゴが認識されると、画面上に 社員からのメッセージ が自動的に表示されます。

---

5. メッセージの閲覧
	- 表示されたメッセージは タップで切り替えできます。

---

6. よくある質問（FAQ）

- Q1. カメラが起動しない／真っ黒のまま
    Safariの設定でカメラが許可されていない可能性があります。
    設定 → Safari → カメラ → 許可 に変更してください。

- Q2. ロゴを読み込んでも何も起きない
    部屋が暗いと認識しづらいことがあります。
    明るい場所でロゴ全体が写るように調整してください。
    ロゴの柄が汚れていたり折れていると認識しにくい場合があります。

- Q3. iPhone以外（Androidなど）でも使える？
    基本的には使用できますが、機種によって動作が異なる場合があります。
    iPhone（Safari）での利用を推奨しています。

---

7. トラブルが解消しない場合
    
    以下をお試しください。
	- Safariを一度終了して再読み込みする
	- iPhoneを再起動する
	- 別の照明環境でロゴを読み込む
	- Wi-Fi切り替え（ネットワーク遅延の可能性）
